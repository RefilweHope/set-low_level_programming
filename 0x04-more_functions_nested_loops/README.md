# 0x04-more_functions_nested_loops

This project focuses on implementing more functions and using nested loops in C. All functions follow the Betty coding style and requirements specified in the project.

## Tasks

- **0-isupper.c** - Check if a character is uppercase
- **1-isdigit.c** - Check if a character is a digit
- **2-mul.c** - Multiply two integers
- **3-print_numbers.c** - Print numbers 0 to 9
- **4-print_most_numbers.c** - Print numbers 0 to 9 except 2 and 4
- **5-more_numbers.c** - Print 0 to 14, ten times
- **6-print_line.c** - Draw a straight line
- **7-print_diagonal.c** - Draw a diagonal line
- **8-print_square.c** - Print a square
- **9-fizz_buzz.c** - The classic Fizz-Buzz program
- **10-print_triangle.c** - Print a triangle

## Compilation

Compile using:
```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 <filename>.c _putchar.c -o <output>
```
