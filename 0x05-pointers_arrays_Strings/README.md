
# C - Low Level Programming

## Description

This repository contains projects completed as part of the ALX Software Engineering programme. Each directory covers a specific topic in low-level programming using the C language, from basic concepts like variables and control flow to more advanced topics like pointers, memory management, and data structures.

## Projects

| Directory | Description |
|-----------|-------------|
| `0x05-pointers_arrays_strings` | Pointers, arrays, and string manipulation in C |

## General Requirements

- All files compiled on Ubuntu 20.04 LTS
- Compiler: `gcc` with flags `-Wall -Werror -Wextra -pedantic -std=gnu89`
- All files must end with a new line
- No global variables
- Maximum 5 functions per file
- Code follows Betty style guidelines (`betty-style.pl` and `betty-doc.pl`)
- All prototypes declared in `main.h`

## Betty Style

All code in this repository follows the Betty coding style for C. To check your code:

```
betty-style.pl <file.c>
betty-doc.pl <file.c>
```
