# 0x0B. C - Preprocessor

## Description
This project covers the C preprocessor — macros, header guards, and
built-in preprocessor macros. It explores object-like macros, function-like
macros, and how the preprocessor works before compilation.

## Learning Objectives
- What macros are and how to use them
- What the most common predefined macros are
- How to include guard a header file
- How to use `#define` for constants and function-like macros

## Requirements
- Allowed editors: `vi`, `vim`, `emacs`
- Compiled on Ubuntu 20.04 LTS with flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- All files end with a new line
- No global variables
- Maximum 5 functions per file
- Code follows Betty style

## Files

| File | Description |
|------|-------------|
| `0-object_like_macro.h` | Defines macro `SIZE` as `1024` |
| `1-pi.h` | Defines macro `PI` as `3.14159265359` |
| `2-main.c` | Prints the name of the file it was compiled from |
| `3-function_like_macro.h` | Defines macro `ABS(x)` to compute absolute value |
| `4-sum.h` | Defines macro `SUM(x, y)` to compute the sum of two numbers |
