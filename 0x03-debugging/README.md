# 0x03. Debugging

##  Description

This project focuses on learning how to **debug C programs**.  
It covers common mistakes such as:

- Infinite loops  
- Incorrect logic in conditional statements  
- Miscalculations in arithmetic and comparisons  
- Handling special cases like leap years  

All programs are written in C and compiled with:

## Project Structure

Directory: `0x03-debugging`

| File | Description |
|------|------------|
| 0-main.c | Tests `positive_or_negative()` function with 0 |
| 1-main.c | Example of an infinite loop, commented out for debugging |
| 2-largest_number.c | Corrected logic to find the largest of 3 integers |
| 3-convert_day.c | Converts a date to the day of the year |
| 3-print_remaining_days.c | Prints remaining days in the year, handles leap years |
| main.h | Header file for all tasks, contains function prototypes |
